Política de Privacidade

Easy Diet — Última atualização: 28 de fevereiro de 2026

O Easy Diet foi desenvolvido por Leandro Costa como um aplicativo de planejamento alimentar personalizado. Esta página informa aos usuários sobre nossas práticas em relação à coleta, uso e proteção de dados pessoais.

🔒 O Easy Diet não coleta, armazena nem transmite seus dados pessoais para servidores externos. Todos os seus dados permanecem exclusivamente no seu dispositivo (iPhone/iPad).

1. Dados Armazenados Localmente

Para funcionar corretamente, o aplicativo armazena as seguintes informações apenas no seu dispositivo:

  • Nome de exibição
  • Peso, altura e idade
  • Objetivo alimentar (emagrecimento, hipertrofia ou manutenção)
  • Localização (cidade e estado) — utilizada uma única vez para personalizar o plano alimentar com alimentos regionais
  • Plano alimentar gerado
  • Histórico de evolução de peso

Essas informações não são enviadas para nenhum servidor e são apagadas quando o aplicativo é desinstalado.

2. Serviços de Terceiros

O Easy Diet utiliza os seguintes serviços de terceiros que podem coletar informações anônimas de uso:

  • Firebase Analytics (Google) — Coleta dados anônimos e agregados sobre o uso do aplicativo (como número de sessões e telas visitadas) para melhorias de qualidade. Nenhum dado pessoal é identificado. Política do Firebase
  • Firebase Crashlytics (Google) — Coleta automaticamente relatórios de erros técnicos (crashes) para melhorar a estabilidade do aplicativo. Política do Firebase
  • RevenueCat — Gerencia assinaturas e compras dentro do aplicativo. Os dados de transação são processados diretamente pela Apple e pelo RevenueCat, sem que o desenvolvedor tenha acesso a dados financeiros pessoais. Política do RevenueCat
  • OpenAI API — Utilizada para gerar planos alimentares personalizados. Os dados enviados (perfil básico sem identificação pessoal) são processados conforme a Política de Privacidade da OpenAI.

3. Localização

O aplicativo solicita acesso à sua localização uma única vez durante a configuração inicial para identificar sua cidade e estado. Essa informação é usada exclusivamente para sugerir alimentos regionais no plano alimentar. A localização é armazenada apenas localmente e não é rastreada continuamente.

4. Dados de Crianças

O Easy Diet não é direcionado a crianças menores de 13 anos e não coleta intencionalmente dados de menores.

5. Segurança

Como todos os dados são armazenados localmente no seu dispositivo, a segurança dos seus dados está protegida pelas medidas de segurança nativas do iOS (criptografia de disco, Face ID/Touch ID, código de acesso).

6. Seus Direitos

Você pode a qualquer momento:

  • Acessar seus dados diretamente no aplicativo
  • Excluir todos os seus dados através do botão "Resetar Todos os Dados" nas Configurações
  • Revogar a permissão de localização nas Configurações do iOS

7. Alterações nesta Política

Podemos atualizar esta Política de Privacidade periodicamente. Recomendamos verificar esta página regularmente. As alterações entram em vigor imediatamente após a publicação.
